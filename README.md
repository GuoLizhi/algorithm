### 项目简介
本项目主要分为两个部分
- 基础数据结构：会使用Java和TypeScript实现底层的数据结构，并考虑数据结构中每个操作的时间复杂度
- leetcode算法：算法主要会记录在本仓库的issue中，leetcode的每一个分类我都以标签的形式标记出来，欢迎参考

> 学习算法是一个漫长的过程，没法靠突击，只能一步一步慢慢积累。
