### Project Introduction
本项目旨在帮助自己学习算法与数据结构，项目中的算法和数据结构均由`typescript`实现。使用`typescript`的主要目的在于它的强类型控制以及一些高级特性比如：泛型，接口。所有的数据结构均手动实现，不会依赖原生JS中的数组方法。并且在原生数组的基础上，扩展实现了一些比较有用方法，比如数组的`remove()`和`add()`。
